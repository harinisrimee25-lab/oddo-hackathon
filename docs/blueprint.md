# **App Name**: AccessKey

## Core Features:

- User Authentication: Securely authenticate users via username/password.
- Password Reset: Allow users to reset their password if they forget it.
- Session Management: Maintain user sessions to keep users logged in.

## Style Guidelines:

- Primary color: Deep purple (#673AB7) to convey security and professionalism.
- Background color: Light lavender (#EDE7F6), a desaturated shade of the primary color for a calm and secure background.
- Accent color: Electric blue (#3F51B5) for interactive elements such as buttons and links.
- Font: 'Inter', a sans-serif font for clean and modern readability across all text elements.
- Use simple, clear icons for user feedback such as success, failure and error states. All icons should consistently use the electric blue accent color.
- The login form should be centered on the page with clear labels and input fields. Employ a single-column layout for ease of use.
- Subtle animations for focus states on input fields and transitions when authenticating.
